### 🪴 My notable pet projects
**A processor from scratch (following Nand2Tetris)**  
I followed The Elements of Computing Systems book course and built a complete computer system from the ground up – starting with NAND gates and ending with a Hack computer's CPU, an assembler, and a compiler for its own high-level language Jack. By the end, I could write a program, compile it through my own compiler, and watch it slowly but surely run on hardware I built.

**A Eurorack synthesizer (following Moritz Klein)**  
I'm currently building my own Eurorack modular synthesizer, following the excellent guides by Moritz Klein. This means designing and soldering analog circuits – oscillators, filters, envelopes – and understanding how voltage control shapes sound. I'm not an electrical engineer, but working through these projects gives me a solid hands-on feel for how hardware and signals work.
